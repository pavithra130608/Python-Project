# Housing Data Analysis using Python

## Project Overview
This project focuses on analyzing housing data to understand pricing trends, factors affecting house prices, and market patterns. The goal is to extract meaningful insights that can help buyers, sellers, and real estate businesses make informed decisions.

---

## Problem Statement
Analyze housing dataset to identify key factors influencing house prices and uncover trends in the real estate market.

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Data Analysis Steps
- Data Cleaning (handling missing values, duplicates)
- Exploratory Data Analysis (EDA)
- Feature Analysis
- Data Visualization

---

##  Key Analysis Performed
- Price distribution of houses
- Relationship between area and price
- Impact of number of bedrooms and bathrooms on price
- Location-based price variations
- Correlation between different features

---

## Visualizations
- Histogram (Price distribution)
- Scatter Plot (Area vs Price)
- Bar Charts (Bedrooms, Bathrooms vs Price)
- Heatmap (Correlation analysis)

---

## Key Insights
- House prices increase significantly with area
- Number of bedrooms and bathrooms directly impacts pricing
- Certain locations have consistently higher prices
- Strong correlation observed between area and price

---

##  Business Impact
- Helps real estate companies understand pricing strategies
- Assists buyers in identifying fair property prices
- Supports market trend analysis for investment decisions

---

##  Conclusion
This project demonstrates how data analysis techniques can be used to uncover trends in housing markets and support better decision-making.
